# Sprint 16 UI 验收清单

## 商品编辑页 SKU 区域

- [ ] 1920×1080 下 SKU 卡片不拥挤，右侧操作不竖排
- [ ] SKU 卡片信息层级清楚（名称 / 状态 / 价格 / 库存 / 毛利摘要）
- [ ] 毛利摘要默认可见（原始成本、原始毛利率、标准损耗后毛利率）
- [ ] 损耗模拟、建议售价、warning 默认收起
- [ ] 点击「查看损耗模拟」后展开详情，再点「收起损耗模拟」可收起
- [ ] 多个 SKU 展开状态互不影响
- [ ] 款式图上传按钮正常
- [ ] 「设为主图」/ 商品卡片主图单选正常
- [ ] 启用开关及说明文案正常
- [ ] 删除款式正常（至少保留一行）
- [ ] stock=0 且启用 → 显示「卖光啦！」
- [ ] isActive=false → 显示「已停用」
- [ ] 移动端 SKU 卡片上下排列可读

## 订单履约看板

- [ ] 待处理 / 制作中 / 配送中等活跃列仍显示完整订单卡片
- [ ] 「已完成 / 历史归档」列显示紧凑卡片
- [ ] 紧凑卡片仅显示：订单号、金额、日期、状态
- [ ] 点击紧凑卡片打开订单详情弹窗
- [ ] 历史订单较多时看板不会被无限拉长
- [ ] 超过 20 条时显示「仅显示最近 20 单」提示
- [ ] 拖拽流转逻辑不受影响（归档列仍不可拖出）

## 供应商表格

- [ ] 供应商名称列左侧冻结
- [ ] 操作列右侧冻结
- [ ] 横向滚动时主列与操作列始终可见
- [ ] 操作按钮（编辑 / 停用）不竖排
- [ ] 样式与物料母表 sticky 列一致

## 其他带操作列表格

- [ ] 商品列表（CMS）：品名左冻结、操作右冻结
- [ ] 采购单列表：采购单号左冻结、操作右冻结
- [ ] 包装方案：名称左冻结、操作右冻结
- [ ] CMS Banner：海报左冻结、操作右冻结
- [ ] CMS 推荐位：名称左冻结、操作右冻结
- [ ] 物料母表：沿用 sticky 规范（参考实现）
- [ ] 编辑 / 停用 / 删除等操作仍可点击

## 移动端

- [ ] SKU 卡片上下排列
- [ ] 宽表格可横向滚动，sticky 列仍可见
- [ ] 看板紧凑卡片可点击打开详情
